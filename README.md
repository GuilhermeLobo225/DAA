# DAA 2025/26 - Grupo MIA
### Tema: Modelação e previsão do fluxo de tráfego rodoviário

## Membros

- João Paulo Martins de Azevedo(PG61693)

## Objetivo
Desenvolver modelos de Machine Learning capazes de prever o fluxo de tráfego numa cidade portuguesa,
usando dados históricos de tráfego fornecidos pela competição Kaggle DAA-TG.
O projeto inclui duas tarefas:
1. **Análise e Validação** — exploração, preparação e análise crítica dos dados e dos resultados obtidos.  
2. **Competição** — desenvolvimento e otimização de modelos preditivos submetidos na plataforma Kaggle.
